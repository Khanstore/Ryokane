Website Review

Odoo Website product review module allows you to implement ratings and review system on your Odoo website. The customers can give star ratings and write a review towards a product on Odoo website.

Information

User Guide
https://webkul.com/blog/odoo-website-product-review/

Basic Features

  • Integrate reviews with your products.
  • Add review feature from odoo website.
  • Admin can manage review from odoo admin panel.
  • Registered user can add review from odoo website.
  • Admin can see all review at one place.
  • Reviews can be like or dislike by the registered user.
  • Admin can control reviews loading on website.
  • Provide responsive view for different devices.
