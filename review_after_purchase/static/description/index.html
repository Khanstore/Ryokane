Website: Product Review After Purchase

Module "Odoo Website Product Review After Purchase" allows your customers to submit reviews from the website. Also with the help of this module automatic mail sent to the customer after purchasing a product, for review submission.

Information

User Guide
https://store.webkul.com/Odoo-Website-Product-Review-After-Purchase.html
Basic Features
  • Add review feature from odoo website.
  • Admin can create multiple Review configuration settings, but only one can be active at a time.
  • Configure Review settings based on order status.
  • Automatic mail will be sent to the customer after product purchase.
  • Admin can manage review from odoo admin panel.
  • Registered user can add a review from odoo website.
